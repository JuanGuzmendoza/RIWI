3. Automatic Digital ID Card Generator for SENA Apprentices

This project automates the creation of digital ID cards for apprentices at the SENA training center. Each ID card includes the apprentice's name, photo, barcode for attendance scanning, and a personalized QR code linking to an individual profile with additional details. The system is built with Laravel and integrates Google's API to use Google Drive as a file system for storing files securely.

Technologies
  • Laravel
  • Google API
